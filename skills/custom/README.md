# 自定义 Skills

这个目录用于放置项目内自定义整理、编写或二次适配的 Skills。

建议补齐这些信息：

- 目标场景
- 能力边界
- 推荐用法
- 已知限制

如果某个 Skill 只适用于单一私有项目，建议不要提交到公共仓库。
