# 通用命令

这个分类用于暂时无法明确归入其他分类、但具有复用价值的命令。

建议只有在下面两种情况下才放入这里：

- 命令确实跨工作流、跨语言通用
- 还没有足够多的同类命令形成独立分类

如果某一类命令逐渐变多，建议从这里拆出新的专题目录，而不是长期堆在 `general/` 中。
